# MCC対応マッピングリファレンス（インデックス）

到達目標からMCCカテゴリへの対応付けに使用するキーワードマッピング。各カテゴリの詳細（到達目標一覧・学習の目安となる項目）は `mcc-categories/` 配下の個別ファイルを参照。

## 使い方

- **キーワード照合・カテゴリ選択**: このファイルのキーワードを使って照合・一覧表示する
- **到達目標の詳細参照**: 該当カテゴリの詳細ファイル（`mcc-categories/xxx.md`）を読む

# プレビュー・出力フォーマット

Step 6 のプレビュー表示および Step 8 のMarkdown出力で使用するフォーマット。

> **契約の正本**: セクション構成（`## 基本情報` / `## 到達目標` / `## ルーブリック` / `## 授業の進め方` / 構造化サブセクション等）・レポート/試験スケジュールの列仕様・名前の正規化ルールは、プロジェクトルートの `specs/syllabus-markdown-schema.md` を参照する。本ファイルはそのスキーマに加えて、プレビュー・Markdown 出力に固有の評価割合・計算方法・計算例テンプレートを記述する。

全体のセクション構成はスキーマ § 1 に従う。下記はスキーマで規定された範囲に加えて、プレビュー出力時に必要なテンプレート要素（評価割合の詳細表・計算例など）を示す。


# 出力ファイル フォーマット仕様

Step 7 のファイル出力で使用するフォーマット。

## 個別フィードバック Markdown

ファイル名: `[学籍番号]_[氏名]_feedback.md`
配置先: `output/[年度]_[教科名]/reports/[レポート名]/feedback/`

```markdown
# レポート評価フィードバック

| 項目 | 内容 |
|------|------|
| 科目名 | [科目名] |
| レポート名 | [レポート名] |
| 採点日 | [YYYY-MM-DD] |
| 学籍番号 | [番号] |
| 氏名 | [氏名] |

---

## 観点別評価

### [観点1名]
- **評価**: [理想的 / 標準的 / 未到達]
- **点数**: [N]点 / [M]点

> 根拠: ルーブリックの「[段階]」の定義「[引用]」に合致。具体的には、[説明]。

### [観点2名]
- **評価**: [理想的 / 標準的 / 未到達]
- **点数**: [N]点 / [M]点

> 根拠: [同上のフォーマット]

...

---

## 総合点

**[N]点 / 100点**

---

## フィードバック

**良い点**: [良い点の記述]

**改善点**: [改善点の記述]

**次回に向けて**: [次のアクション]
```

### チーム提出の場合

基本情報テーブルに以下を追加:

```markdown
| チーム名 | [チーム名] |
| メンバー | [氏名1], [氏名2], [氏名3] |
```

フィードバックファイルはチームで1つ生成する。ファイル名は `[チーム名]_feedback.md`。

---

## 採点一覧 Markdown

ファイル名: `[レポート名]_採点一覧.md`
配置先: `output/[年度]_[教科名]/reports/[レポート名]/`

```markdown
# 採点一覧: [レポート名]（[科目名]）

| 項目 | 内容 |
|------|------|
| 科目名 | [科目名] |
| レポート名 | [レポート名] |
| 採点日 | [YYYY-MM-DD] |
| 採点済み | [N]名 |
| 未提出 | [N]名 |

## 成績サマリー

| 統計 | 値 |
|------|-----|
| 平均点 | [N]点 |
| 最高点 | [N]点 |
| 最低点 | [N]点 |
| 中央値 | [N]点 |

## 採点結果一覧

| # | 学籍番号 | 氏名 | [観点1] | [観点2] | ... | 合計 | 備考 |
|---|---------|------|---------|---------|-----|------|------|
| 1 | [番号] | [氏名] | [N]点 | [N]点 | ... | [N]点 | |
| 2 | [番号] | [氏名] | [N]点 | [N]点 | ... | [N]点 | |
| ... | ... | ... | ... | ... | ... | ... | ... |

## 観点別 評価分布

| 評価観点 | 理想的 | 標準的 | 未到達 | 平均点 |
|----------|--------|--------|--------|--------|
| [観点1] | [N]名 | [N]名 | [N]名 | [N]点 |
| [観点2] | [N]名 | [N]名 | [N]名 | [N]点 |
```

---

## フォーマット規則

1. **エンコーディング**: UTF-8、BOMなし
2. **セクション順序**: 基本情報 → 観点別評価 → 総合点 → フィードバック の順で固定する
3. **採点根拠**: 必ず引用ブロック（`>`）で記載する

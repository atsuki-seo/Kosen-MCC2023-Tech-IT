## 到達目標

・プログラムで扱うデータの特性に応じた適切なデータ構造の選択は、効率的な問題解決に不可欠である。
・本講義では、基本的なデータ型・データ構造と代表的なアルゴリズムについて学び、以下の能力の習得を目標とする。
　・基本的なデータ構造（配列、スタック、キュー、リスト、木構造）の特徴を理解し、目的に応じて選択・活用できる
　・代表的な探索・ソートアルゴリズムの手順と特徴を理解し、実装できる
　・アルゴリズムの計算量（O記法）を理解し、効率を評価できる

## 概要
・プログラムで扱うデータの特性に応じた適切なデータ構造の選択は、効率的な問題解決に不可欠である。
・本講義では、基本的なデータ型・データ構造（配列、スタック、キュー、リスト、木構造）と、代表的な探索・ソートアルゴリズムについて解説する。
・学んだアルゴリズムの応用としてvsCPU形式のリバーシプログラムの開発に取り組み、探索アルゴリズムを実践的に活用できる能力の習得を目標とする。

## 授業の進め方
・毎回の授業は前半30分を講義、後半60分を演習に充てる。
・教材はMoodle（https://moodle2026.center.yuge.ac.jp）で配布する。
・小テストは単元ごとに3回実施する（選択式、ノート類の持ち込み可、AI/ウェブ検索の使用を許可するが短時間で解答する前提）。
　・第5週: 小テスト①（範囲: 第1〜4週）
　・第8週: 小テスト②（範囲: 第5〜7週）
　・第10週: 小テスト③（範囲: 第8〜9週）
・期末試験は第15週に実施し、全範囲（第1〜14週）を対象とする。
・後半の授業（第11〜14週）ではvsCPU形式のリバーシプログラムの開発に取り組む。

## 注意点
・演習で使用するソースコードや資料はMoodleで提供するので、授業時間外でも動作確認や復習を行うこと。
・小テストは短時間で実施するため、各単元の内容を事前に整理しておくこと。
・リバーシプログラムの開発では、実装したコード・動作検証用のコード、AI利用の場合は投入したプロンプト・やり取りの全ログがわかるテキストまたはスクリーンショットを提出すること。

## 授業計画
| 週 | 授業内容 | 到達目標 |
|----|----------|----------|
| 1週 | ガイダンス・基本データ型 | 基本データ型の種類と特徴を理解し、用途に応じた使い分けを説明できる。 |
| 2週 | 変数・配列 | 変数と配列の仕組みを理解し、プログラムで活用できる。 |
| 3週 | スタック・キュー | スタックとキューの構造・操作を理解し、実装できる。 |
| 4週 | リスト（線形・双方向・カーソル） | 各種リスト構造の特徴を理解し、実装できる。 |
| 5週 | 木構造（二分木・探索木）・小テスト①（範囲: 第1〜4週） | 木構造の概念と二分探索木の操作を理解し、実装できる。 |
| 6週 | 計算量・アルゴリズム評価 | O記法を用いてアルゴリズムの計算量を評価・比較できる。 |
| 7週 | 線形探索・二分探索 | 線形探索と二分探索の手順・特徴・計算量を理解し、実装できる。 |
| 8週 | ハッシュ探索・小テスト②（範囲: 第5〜7週） | ハッシュ法の仕組みを理解し、衝突解決法を含めて説明できる。 |
| 9週 | ソートアルゴリズム①（バブル・選択・挿入） | 基本的なソートアルゴリズムの手順と計算量を理解し、実装できる。 |
| 10週 | ソートアルゴリズム②（シェル・クイック・ヒープ）・小テスト③（範囲: 第8〜9週） | 高速なソートアルゴリズムの手順と計算量を理解し、実装できる。 |
| 11週 | ゲーム木探索（ミニマックス法）・リバーシプログラム導入 | ゲーム木探索の考え方を理解し、リバーシへの適用方針を説明できる。 |
| 12週 | リバーシプログラム開発①（盤面管理・合法手生成） | リバーシの盤面をデータ構造で表現し、合法手の判定を実装できる。 |
| 13週 | リバーシプログラム開発②（探索・評価関数の実装） | 探索アルゴリズムと評価関数を実装し、CPUの思考ロジックを構築できる。 |
| 14週 | リバーシプログラム提出・振り返り | 完成したプログラムを提出し、使用したアルゴリズムの選択理由を説明できる。 |
| 15週 | 期末試験（範囲: 第1〜14週） | — |
